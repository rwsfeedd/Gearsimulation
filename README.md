GearSimulation  
----------------------------------------------------------------------------------------------
Program made to potentially simulate multiple Gears and the interaction with and between them.

Programacquisition:
cmake with make:
To load this Program onto your Computer, you have to copy the files from the build directory into one of your one directories.
Then you have to run "cmake ." in the choosen directory, followed by "make".
After that you can execute the Programm with "./Gearsimulation".
